## Maintenance & Consumables

### Fluids & Filters
- [ ] Motorcraft AGSF32PM Spark Plugs — OEM spec, pre-gapped 0.028", replace every 30k mi (4 plugs)
- [ ] Motorcraft FL-910S Oil Filter — OEM spec
- [ ] Motorcraft 5W-30 Full Synthetic — 5.7qt capacity per change
- [ ] Ford BM5Z-9601-A OEM Air Filter — or K&N 33-2435 for reusable
- [ ] Ford DG9Z-19N619-A Cabin Air Filter — replace annually
- [ ] Motorcraft VC-3-B Orange Coolant — 50/50 premix, ~2.5qt top-up capacity
- [ ] DOT 4 LV Brake Fluid — Ford spec WSS-M6C65-A2, ATE or Motul RBF600
- [ ] Motorcraft Full Synthetic MTF — MT82 gearbox, 1.8L fill, reduces crunch in cold weather
- [ ] Motul 8100 X-clean+ 5W-30 — popular OEM+ oil upgrade

### Wear Items
- [ ] [Ford OEM Key Fob](https://www.amazon.com/s?k=164-R8070+ford+focus+key+fob) — part# 164-R8070, 4-button 315MHz, includes remote start button
- [ ] Key Fob Battery — CR2032, replace if range drops before buying new fob
- [ ] Key Fob Programming — self-program: cycle ignition 8× in 10s until chime, press any button within 8s; or dealer ~$75–100
- [ ] [KeylessOption Aftermarket Fob](https://www.amazon.com/s?k=keylessoption+ford+focus+st+key+fob) — ~$15, needs programming, same 315MHz
- [ ] Spare Transponder Key — get a second key cut + programmed (dealer or locksmith ~$100–150 total)
- [ ] Front Brake Pads — 320mm front, replace around 30k mi depending on use
- [ ] Rear Brake Pads — 302mm rear
- [ ] Front Rotors — stock 320mm vented
- [ ] Rear Rotors — stock 302mm solid
- [ ] Drive Belt — Ford 6PK1763 serpentine, inspect at 60k
- [ ] Clutch Kit — stock clutch holds ~280ft-lb, upgrade if tuned
- [ ] OEM Wiper Blades — 26" driver / 16" passenger (Bosch Icon fit well)
- [ ] Thermostat — replace if running warm, common weak point on EcoBoost
- [ ] PCV Valve — inspect at 60k, clogging causes oil leaks
- [ ] Spark Plug Coil Pack Set — replace with plugs if misfiring at 60k+

### Tools
- [ ] Cobb AccessPort V3 — also functions as OBD2 scanner / datalLogger
- [ ] Admin Key — check dash: Settings → MyKey → Admin Key Detected. If 0 admin keys, MyKey restrictions are active (speed cap, volume limit, etc.)
- [x] OBDLink MX+ — Bluetooth adapter, full MS-CAN + HS-CAN support, works with FORScan
- [ ] [FORScan](https://forscan.org/download.html) — free, pair with MX+ over Bluetooth. Resets MyKey, unlocks hidden config (disable fake engine noise, EPAS steering weight, hidden dash menus, all module codes)
- [ ] Admin Key Reset via FORScan — BCM → MyKey → Reset MyKey, makes current key the admin. Free vs dealer $75–100
- [ ] Autel MaxiLink ML619 — budget OBD2 scanner with ABS/SRS codes
- [ ] Oil Drain Pan — 8qt+ for 5.7qt capacity
- [ ] Torque Wrench — 0-150 ft-lb range, needed for wheel bolts (103 ft-lb) + drain plug (20 ft-lb)
- [ ] T30 Torx Bit Set — brake caliper slider pins
- [ ] 32mm Socket — front hub nut (one-time use if replacing wheel bearing)
- [ ] Oil Filter Wrench — fits FL-910S
- [ ] Jack Stands (x4) — never use just a floor jack
